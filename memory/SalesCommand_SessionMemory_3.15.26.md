## Core Architecture
- Single-file HTML/CSS/JS — no framework, no build step, no external deps except Google Fonts
- All rendering via string-based `innerHTML` injection — no virtual DOM
- Central `render()` dispatcher — switch on `state.page`, calls page renderer, injects into `#content`
- All renderers return HTML strings
- State mutations always call `buildNav(); render()`
- `state` is the single mutable runtime object — everything flows through it

## State Object Shape
```js
let state = {
  page, role, selectedRep, selectedDeal, dealFilter, alertFilter,
  aiContext, repViewAs, chatMessages, chatTyping,
  copilotChats, copilotTyping, copilotApiKey, copilotTabs,
  accountIntelligence
}
```

## Copilot Tab System
Tab state in `state.copilotTabs[dealId]`. Values: `assessment | discovery | intelligence | plan | drafts`
- Assessment: `buildPipelineCopilot(d)` — situation summary + coaching actions + AI chat
- Guided Discovery: 15 questions with prerequisite chain — answers in `state.accountIntelligence[dealId]`
- Deal Intelligence: `buildDealIntelligenceModel(d)` — 4-quadrant read-only synthesis
- Messaging Plan: 3-contact outreach plan — requires ≥3 Discovery answers
- Drafts: email + call plan + manager brief — copy buttons

## Alert System
`generateAlerts()` runs on init + after every mutation. Severity: `critical → high → medium → low`. Alerts are dismissable. Nav badge = active count.

## Sandler Field Templates (for reference when editing guidance text)
- **Pain:** Current problem / Where it shows up / Who is affected / Urgency trigger / Prospect quote
- **Impact:** Financial / Operational / Strategic / Cost of doing nothing / Evidence
- **Budget Range:** Status / Range / Source / Competing priorities / Confidence level
- **Economic Buyer:** Name+title / How confirmed / Access level / Motivations / Risk of going around
- **Decision Process:** Steps to signature / Stakeholders / Procurement+legal / Timeline / Blockers
- **Decision Timeline:** Target date / Driver / Flexibility / Budget cycle / Urgency signal
- **Champion (Sandler):** Name+role / Motivation / Access to power / Tested commitment / Risk
- **Mutual Next Step:** What / Who owns it / Deadline / Confirmed by prospect / Consequence if missed
